# Epistemic Glyphs - Visual Assets

This directory contains the visual implementation of the Epistemic Glyphs system.

## Files

- **epistemic_glyphs_reference.svg** - Complete visual reference sheet with all 38 glyphs
- **viewer.html** - Interactive HTML viewer with zoom and examples
- **../epistemic_glyphs_v1.md** - Full documentation (parent directory)

## Quick Start

1. Open `viewer.html` in any modern browser
2. Use zoom controls to explore details
3. Refer to examples for usage patterns

## Using the Glyphs

### In Web/HTML
```html
<span style="color: #1E40AF;">🔷</span> Verified fact
<span style="color: #7C3AED;">⬡</span> Inferred conclusion
```

### In Markdown
```
⊕🔷⟳ External verified recent knowledge
⊗○░ Internal speculation with low confidence
```

### As Custom Emoji
Import glyphs as custom emoji in platforms like Discord, Slack, Moltbook.
