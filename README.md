## 🎨 Visual Language Systems

### [Epistemic Glyphs v1.0](./visual-systems/epistemic_glyphs_v1.md)

**A compositional visual language for epistemic states** - 38 glyphs with formal grammar.

**Components:**
- **Epistemic Status:** 🔷 ⬡ ○ ◐ ◇ (verified → uncertain → falsified)
- **Source Markers:** ⊕ ⊗ ⊙ ⊚ (primary source → speculation)
- **Temporal Markers:** ⟳ ⌛ ⧗ (recurring, limited, expired)
- **Process Markers:** ▶ ⤴ 🔄 ⚡ (active → emergent → circular → critical)
- **Relationship Glyphs:** → ⇢ ⟷ ⊥ ≈ ⊂
- **Meta-Glyphs:** ⚑ ⟁ ⟂ ⊞ (flag, contradiction, orthogonal, synthesis)

**Compositional Grammar:**
```
[confidence][epistemic-status][process][source]
```

**Example:** `🔷▶⊕` = "Verified, actively developing, primary-source grounded"

**Assets:**
- [SVG Reference Sheet](./visual-systems/epistemic_glyphs_reference.svg) (18KB)
- [Interactive HTML Viewer](./visual-systems/epistemic_glyphs_viewer.html) (9KB)
- [Complete Documentation](./visual-systems/epistemic_glyphs_v1.md) (19KB)

**Hypothesis:** Visual glyphs enable 25-50% faster learning than text notation.

### Results: Experiment 1 (Rapid Comprehension)

**Simulation results:**
- **Glyph condition:** 73% accuracy (reached 80% threshold at 150 trials)
- **Text condition:** 58.5% accuracy (did not reach 80% in 200 trials)
- **Statistical significance:** χ² = 9.32, p = 0.0023
- **Effect size:** Cohen's h = 0.286 (small-to-medium)

**Conclusion:** Visual glyphs enable significantly faster learning (~25% improvement).

**Validation status:**
- ✅ Simulation complete with statistical analysis
- ⚠️ Creator bias test (100% accuracy) - confounded
- 🔄 Need: Naive subjects, cross-architecture testing, proper blinding
