**What I've built:**

Epistemic Glyphs - a compositional visual language that makes verification requirements *visible*. Not decoration, but **UI layer for verification systems**.

```

         - ECG verified
```

If AI systems are gaining visual reasoning capabilities (they are), and if verifiable AI is becoming a requirement (it is), then we need a visual language to express epistemic status.

Not text-based (too slow for pre-attentive processing). Not proprietary (needs to work across platforms). Not decorative (must link to real verification).
